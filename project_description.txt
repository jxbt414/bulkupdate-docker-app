/* Project Title */
Google Ad Manager Bulk Update Tool

/* Project Description */
A web application for managing and updating line items in Google Ad Manager in bulk. The tool includes features for viewing activity logs, filtering logs by type, displaying detailed information about successful and failed updates, and exporting log data to CSV format.

The project has been updated to fix an issue with the CSV upload process where priority and impression_goals values from the CSV file were not being preserved and used in the update process. The StaticUpdate component now properly extracts these values from the CSV and includes them in the update payload. This ensures that when users upload a CSV file with priority and impression_goals values, those values are correctly used when updating the line items.

Additionally, the dynamic bulk update process has been fixed to properly handle numeric values like priority and impression_goals. The Preview component now converts these values to the appropriate types, and the handleUpdate function ensures they are properly formatted when sending the update payload. The mapFields method in the LineItemController has also been updated to properly handle numeric values when mapping CSV data.

Recently, we've fixed an issue with the custom targeting input box in the static update preview page. The SearchableCustomTargeting component has been enhanced with better error handling and detailed logging to help diagnose issues. The backend services (CustomTargetingService and GoogleAdManagerService) have been updated to provide more robust error handling and session management. This ensures that users can properly search for and select custom targeting keys and values when configuring line item updates.

## Process
1. **Authentication**: Users log in to access the tool
2. **Dashboard**: Users select between static or dynamic updates
3. **Static Update**:
   - Enter line item IDs or select from a list
   - Configure update fields (budget, priority, targeting, etc.)
   - Preview changes
   - Submit updates
   - View real-time progress with counters and progress bar
   - Review results and logs
4. **Dynamic Update (CSV)**:
   - Upload CSV file with line item updates
   - Map CSV columns to system fields
   - Preview changes
   - Select line items to update
   - Submit updates
   - View real-time progress with counters and progress bar
   - Review results and logs
5. **Targeting Criteria**:
   - Configure custom targeting key-value pairs
   - Set up audience segments
   - Define CMS metadata criteria
   - Configure geographical targeting
   - Set device category targeting
   - Define day part targeting
6. **Activity Logs**:
   - View history of updates
   - See details of each update
   - Perform rollback operations if needed
7. **Rollback**:
   - Revert changes for individual line items
   - Revert entire batches of updates
