# Bulk Update Tool for Google Ad Manager Line Items

This tool allows users to bulk update line items in Google Ad Manager through a user-friendly web interface. The system provides the following functionality:

- **CSV Upload**: Users can upload a CSV file containing line item updates
- **Field Mapping**: Users can map CSV columns to system fields
- **Preview**: The system displays current values and new values for each line item
- **Selection**: Users can select which line items to update
- **Update**: The system applies updates to selected line items via the Google Ad Manager API
- **Logging**: The system logs all updates and provides rollback capabilities

## Technical Details

- **Backend**: Laravel PHP
- **Frontend**: React with DaisyUI and Tailwind CSS
- **API Integration**: Google Ad Manager API v202411
- **Data Processing**: CSV parsing, validation, and transformation
- **Error Handling**: Robust error handling with detailed logging
- **Security**: Authentication and authorization

## Key Features

- **CSV Validation**: Validates CSV files for required fields and proper formatting
- **Field Mapping**: Maps CSV columns to system fields with case-insensitive matching
- **Preview Generation**: Shows current values and new values for each line item
- **Bulk Update**: Updates multiple line items at once
- **Rollback**: Allows rolling back updates if needed
- **Logging**: Tracks all updates and errors
- **User Interface**: Clean, responsive UI with progress tracking
